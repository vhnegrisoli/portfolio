Sobre mim

Sou um Desenvolvedor Back-End e um analista de dados, focado em Business Intelligence (BI) e Data Science, e totalmente apaixonado por novas tecnologias.

Apaixonado por tecnologias

Desde que entrei no mundo da tecnologia eu busco estar cada vez mais atualizado e compreendendo as novas tendências do mercado de Back-End e análise de dados, dentre minhas habilidades, posso citar algumas, como:

  • Conhecimento e experiência em arquitetura REST e Microsserviços
  • Back-End com Java 11, Spring Boot, NodeJs, Express e Nest.js
  • Bancos de dados Oracle 11g e 12c, PostgreSQL e Microsoft SQL Server
  • Mensagens com RabbitMQ, QueryDsl para consultas avançadas
  • Linguagem de programação Javascript para Back-End com NodeJs e Front-End com ReactJs e Redux
  • Tratamento de dados com linguagem SQL e Python 3
  • Análise de dados avançada com SQL e Python
  • Softwares de BI como Microsoft PowerBI, QlikView e QlikSense
  • ETL em linguagem de script com SQL, Pandas, Qlik Analysis
  • Data Science com Python 3, Pandas, NumPy, Matplotlib e SQL
  • Criação de dashboards analíticas e gerenciais
